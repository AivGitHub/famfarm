## About The Project

Project for family photos and memories.

## Getting Started

### Installation

1. `python -m venv .venv`
2. `source .venv/bin/activate`
3. `pip install -r requirements.txt`
4. Download bootstrap5 CSS and JS inside `utils/static/base/bootstrap/`
